# 🎙️ MicStream

Приложение для трансляции микрофона с Android телефона на ПК по Wi-Fi. Телефон становится беспроводным микрофоном — звук идёт в реальном времени и доступен любой программе на ПК (Discord, OBS, Teams и др.).

---

## Как это работает

```
Телефон (APK)
    └─ захватывает микрофон
    └─ ищет сервер в локальной сети (HTTP сканирование)
    └─ подключается по WebSocket (WSS)
         │
         ▼
ПК — Electron приложение
    └─ встроенный HTTPS/WSS сервер (порт 3000)
    └─ получает PCM аудио с телефона
    └─ воспроизводит через выбранное устройство
         │
         ▼
VB-Audio Cable (опционально)
    └─ CABLE Input ← MicStream выводит сюда
    └─ CABLE Output ← Discord/OBS видит как микрофон
```

Связь идёт только по локальной сети Wi-Fi. Интернет не используется.

## Требования

### ПК (для запуска готового EXE)
- Windows 10 / 11
- Больше ничего не нужно — Node.js уже встроен в приложение
- Для использования как микрофон в Discord/OBS: [VB-Audio Virtual Cable](https://vb-audio.com/Cable/) (бесплатно)

### ПК (только для сборки из исходников)
- Node.js 18+ — [nodejs.org](https://nodejs.org)

### Телефон
- Android 8.0+
- Оба устройства должны быть в одной Wi-Fi сети

## Сборка ПК приложения (EXE)

```bash
npm install
npm run build
```

Готовый portable EXE появится в папке `dist\`.

> **Важно:** при первой сборке скачивается Electron (~100 МБ). Последующие сборки быстрее.

## Сборка Android APK

### Требования
- [Android Studio](https://developer.android.com/studio) с JDK 17+
- Node.js 18+

### Шаги

1. Открой Android Studio
2. **File → Open** → выбери папку `android/android`
3. Подожди пока Gradle синхронизируется (~2-5 минут первый раз)
4. Собери APK: **Build → Build Bundle(s) / APK(s) → Build APK(s)**
5. APK появится по пути:
   ```
   android/android/app/build/outputs/apk/debug/app-debug.apk
   ```

### Установка APK на телефон
1. Перенеси APK на телефон (USB, Telegram, Google Drive)
2. Настройки → Безопасность → **Установка из неизвестных источников** → Включить
3. Открой APK и установи

## Использование

### Первый запуск
1. Запусти `MicStream.exe` на ПК
2. Открой MicStream на телефоне
3. Телефон автоматически найдёт ПК в сети и подключится
4. Микрофон включится автоматически

### Для Discord / OBS / Teams
1. Установи [VB-Audio Virtual Cable](https://vb-audio.com/Cable/)
2. В MicStream (ПК) нажми шестерёнку → выбери **CABLE Input** как устройство вывода
3. В Discord/OBS выбери **CABLE Output** как микрофон

### Управление
- **Нажать на центр экрана** (телефон) — включить/выключить микрофон
- **Шестерёнка** — настройки: статус подключения, устройство вывода, громкость
- **Свернуть окно** — приложение уходит в системный трей и продолжает работать
- **Двойной клик на иконку в трее** — открыть окно

## Частые вопросы

**Телефон долго ищет ПК**
— Убедись что оба устройства в одной Wi-Fi сети. После первого подключения следующие будут мгновенными.

**Нет звука на ПК**
— Кликни на окно приложения чтобы активировать AudioContext. Проверь громкость в настройках.

**Микрофон не виден в Discord**
— Установи VB-Audio Cable и выбери CABLE Input в настройках MicStream.

**APK не устанавливается**
— Включи "Установка из неизвестных источников" в настройках Android.

## Технологии

- **Electron** — ПК приложение
- **WebSocket (WSS)** — передача аудио
- **PCM Float32** — формат аудио без потерь
- **selfsigned** — самоподписанный SSL сертификат для локальной сети
- **Android WebView** — интерфейс мобильного приложения
- **Web Audio API** — захват и воспроизведение звука
